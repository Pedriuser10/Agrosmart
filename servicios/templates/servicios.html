¿Cómo funciona?

El sistema AgroSmart opera con un flujo continuo desde el sensor hasta tu celular

Placa ESP32 — Nodo central

El microcontrolador ESP32 recibe los datos de los sensores de humedad y temperatura, procesa la información y decide cuándo activar o cortar el riego automáticamente. Opera en tres modos: Automático, Manual y Offline.

Flujo completo del sistema

Los sensores capturan datos en terreno → el ESP32 procesa → el gateway transmite a la nube → el dashboard muestra el estado → las electroválvulas ejecutan el riego. Todo automático, sin intervención manual del agricultor.

Modos de operación

El sistema se adapta a cualquier condición del terreno

Automático

El sistema activa el riego al detectar umbrales críticos de humedad. Sin intervención del usuario.

Manual

El agricultor controla las válvulas de forma remota desde su celular en cualquier momento.

Offline

Ante pérdida de señal, el sistema ejecuta rutinas de riego seguras preprogramadas localmente.
